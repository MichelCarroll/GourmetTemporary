Getting started

  1. Unzip your downloaded thetooltip.zip file.
  2. Follow the steps for integration in a HTML website or a Wordpress website.

Integration in a HTML website

Linking the CSS file

  1. Place thetooltip.css in the directory of your website or project.
  2. Open your HTML document.

  3. Copy the following code:

    <link rel="stylesheet" type="text/css" href="thetooltip.css" />
  4. Paste it within the head tags of your HTML document.

Creating a tooltip

  1. Copy the following code:

    <a href="">anchor<span class="tooltip applegreen">The Tooltip works!</span></a>
  2. Paste it somewhere within the body tags of your HTML document.
  3. See the classname overview to change the classnames to your desire.

Integration in a Wordpress website

Linking the CSS file

  1. Locate your Wordpress theme's directory. By default this is located at wordpress/wp-content/themes/themename/.
  2. Place thetooltip.css inside this directory.
  3. Open header.php.

  4. Copy the following code:

    <?php wp_enqueue_style('thetooltip',get_bloginfo('stylesheet_directory') . '/thetooltip.css'); ?>
  5. Paste it within the head tags of this document.

Creating a tooltip

  1. Login to your Wordpress admin panel.
  2. Open the post or page to which you want to add a tooltip.
  3. Make sure the HTML tab (next to the Visual tab) is activated in the text editor.

  4. Copy the following code:

    <a href="">anchor<span class="tooltip applegreen">The Tooltip works!</span></a>
  5. Paste it in the text editor where you want a tooltip to appear.
  6. See the classname overview to change the classnames to your desire.

Positioning

The Tooltip can be positioned by adding one of the following classnames to it:

  • left
  • center
  • right

By default The Tooltip is positioned left. In this example we are positioning The Tooltip in the center by adding the center classname to it:

<a href="">anchor<span class="tooltip center applegreen">This is a tooltip with center positioning</span></a>

Changing the default width

The Tooltip's default width is 300px. In this example, we are going to change the width from 300px to 500px:

  1. Open thetooltip.css.
  2. Locate .tooltip.

  3. Change

    width:300px;

    to

    width:500px;
  4. To ensure our tooltip centers when we apply the center classname, we need to do one additional adjustment.
  5. Locate .tooltip.center.

  6. Change

    margin-left:-165px;

    to

    margin-left:-265px;

    This is the formula we use to calculate margin-left:

    margin-left = - (width / 2) + padding

    The Tooltip's padding is 15 by default.

  7. Save thetooltip.css.

Using The Tooltip as a speech bubble

By adding the speechbubble classname to your tooltip you can create a speech bubble that's visible by default. Example:

<div class="tooltip speechbubble applegreen">This speech bubble is visible without a hover</div>

Note that speech bubbles are not restricted to div elements. You could do the same with other elements.
